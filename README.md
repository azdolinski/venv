### Install
```bash
wget https://raw.githubusercontent.com/azdolinski/venv/main/venv.sh
```

### Usage
```bash
# --- Usage ---
# sh venv.sh 
Usage: venv.sh {create|activate|deactivate|remove}

# --- Activate ---
# ./venv.sh active
# sh venv.sh active

# --- Deactivate ---
(venv) # deactivate
#

# --- Remove ENV ---
# sh venv.sh remove
Virtual environment removed.

```
